Console Interface Documentation

display(object displayObj)

  Displays object a in the window.

  displayObj: Object to be displayed.

get(string infoType[, bool longText])

  Returns object containing relevant information.

  infoType: The type of information.

  longText: If true, returns long version of information. Defaults to false.

list()

  Displays the types of information that are available.

send(string message)

  Sends me a message [not implemented yet].

  message: Message to be stored.

secret(string command)

  Executes a secret command.

  command: The command to be executed.


Example:

  var originInfo = get("origin");

  display(originInfo);
